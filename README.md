# Comment_Reward_System
채택 댓글 보상 시스템 (금융블록체인서비스특론)

### Tools
- MAMP (MySQL)
- Ganache
- React
- Node.js

### Version
- Solidity 버전
```
-	VScode 확장프로그램인 Solidity : 0.8.10
-	truffle compile : 0.6
-	truffle에 내장된 Solidity : v0.5.16 (solc-js)
```
- node 버전 : v14.19.0
- npm 버전 : 6.14.16
- Truffle 버전 : v5.5.14
- Ganache 버전 : v^7.1.0
- Web3.js 버전 : v1.5.3

### Commands
- Back
```
cd server
node index.js
```

- Front
```
cd client
npm run start
```
